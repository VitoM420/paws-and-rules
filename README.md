# 🐾 Paws & Rules

### *When Home Has Conditions*

An advocacy website highlighting the struggles of pet owners who face strict rules in residential estates and sectional title schemes. Built to amplify real stories, expose systemic issues, and call for legislative reform.

## 🚀 Getting Started

No build tools or installations required. This is a plain HTML/CSS/JS website.

### View Locally
1. Download or clone this repository
2. Open `index.html` in any web browser

### Deploy via GitHub Pages
1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set Branch to `main` and folder to `/ (root)`
4. Click **Save** — your site will be live within a few minutes

## ✏️ Customisation

To update the site, open `index.html` in any text editor and look for these sections:

- **Stories** — Search for `story-card` to edit testimonials
- **Stats** — Search for `stat-num` to update figures
- **Demands** — Search for `demand-card` to edit policy asks
- **Colours** — Edit the `:root` CSS variables at the top of the `<style>` block
